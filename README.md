# 基于多Agent的“无人机-防空阵地”协同反导系统设计

## 📖 项目简介
本项目针对现代高并发、异构饱和攻击背景下的防御调度难题，构建了一种基于多智能体协同的防空系统离散事件仿真框架。项目在物理层面构建了包含前沿巡飞无人机与后方防空阵地的异构拦截网络；在算法逻辑层面，引入 BDI（信念-愿望-意图）认知模型与双层合同网协议（CNP），将传统的全局集中式运筹优化解耦为底层节点的分布式动态竞标过程，有效消除了多节点的火力重叠与漏防死锁问题。

本项目代码按消融实验的逻辑进行组织，逐步递进验证系统架构的费效比优势。

## 📂 目录结构与模块说明

项目核心代码主要分为以下几个递进的仿真阶段及系统调度模块：
```text
├── step1/                                  # 消融实验阶段一：单层独立防御基线
├── step2/                                  # 消融实验阶段二：引入前沿异构实体
├── step3 and external comparison.py        # 最终协同模型与基线算法对比
└── Evernt_Scheduler.py                     # 离散事件调度与系统量化核心引擎
